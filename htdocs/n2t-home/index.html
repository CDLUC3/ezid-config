The Name-to-Thing (N2T) Resolver

 

The N2T resolver is a tiny web server that takes identifiers and returns things related to them.

It keeps the name of a thing persistent by forwarding requests to servers that know about it.

You can ask for the thing itself (a document, dataset, book, etc.) or its description (metadata).

N2T doesn't care if the thing is a statue, specimen, vocabulary term, or symphony orchestra.

N2T also doesn't care if the name is a URL, ARK, DOI, URN, Handle, LSID, etc.

 

EZID is the main way to add to N2T's database of names, descriptions, and forwarding instructions.

N2T is a work in progress, and resolver replication is a high priority (see original N2T vision).
